運動前 · 中 · 後：資源整合架構

平台不只是「找教練」，而是覆蓋完整運動生命週期，各階段對接不同產業夥伴。

運動前 Before

  • 知識與教育：入門觀念、適性評估、健康自評
  • 書籍／課程：運動科學、訓練方法、心理建設
  • 經驗分享：國手、教練、資深玩家、社群前輩
  • 裝備選擇：教練＋專業店導引（非比價電商）
  • 健康篩檢：健檢、運動醫學評估、慢病諮詢

平台角色：知識庫、講座、顧問媒合、企業 ESG 健康講習

運動中 During

  • 教練／道館：技術、陪練、課表
  • 運動防護：防護員、肌貼、護具、熱身流程
  • 場地與賽事：場館、路線、賽事執行
  • 即時安全：保險、緊急 SOP、天候／強度分級
  • 穿戴與數據：心率、配速、訓練紀錄（選配）

平台角色：預約、活動管理、防護人力媒合、贊助現場服務

運動後 After

  • 飲食與營養：恢復餐、運動營養師諮詢
  • 物理治療／復健：痠痛、傷害評估、復健轉介
  • 醫藥保健：運動醫學、預防性衛教（非推銷用藥）
  • 補給：電解質、蛋白、恢復用品（標示「輔助」）
  • 休息與睡眠：恢復科學、過訓練預防

平台角色：轉介網絡、恢復課程、品牌體驗（受使命護欄約束）

全球化策略：台灣 → 亞洲 → 全球

Phase A · 台灣（第 1～2 年）

  • 1～2 城市深度試點，2～5 運動分館
  • 串接在地：教練、防護、物理治療、保險、出版社
  • 建立「前中後」SOP 與資料模板（中英雙語雛形）
  • 與運動部、國健署、地方政府試點合作
  • 目標：可複製的 impact 報告與贊助 package

Phase B · 亞洲（第 2～4 年）

  • 優先：日本、韓國、東南亞華語圈、香港
  • 模式輸出：平台白標、教練認證、賽事 IP 授權
  • 連結亞洲品牌贊助、跨國賽事、運動醫學交流
  • 多語系：繁中、簡中、英、日（依市場）
  • 目標：3～5 個亞洲合作節點

Phase C · 全球（第 4～6 年）

  • 輸出「運動健康作業系統」：Know-how + 軟體 + 認證
  • 參與國際運動健康、Exercise is Medicine 網絡
  • 全球品牌長約贊助、線上內容訂閱
  • 與國際出版、運動科學機構內容授權
  • 目標：使命型平台成為亞洲標竿，再拓全球華人與英文市場

組織擴張：快速整合產業脈絡的人力架構

健康倫理與利益衝突（醫藥／補給特別注意）

  • 平台使命是運動促進健康，醫藥與補給僅能作「輔助與衛教」，不得替代運動或鼓吹過度用藥。
  • 物理治療、運動醫學內容須由合格專業人員審核或署名。
  • 補給品牌贊助須標示「無法取代均衡飲食與規律運動」。
  • 轉介物理治療／診所時揭露合作關係，避免不當導流。
  • 書籍、知識內容標明作者資格與更新日期。
與「預防勝於治療」一致的合作： 保險公司（預防保健）、健檢中心（運動前篩檢）、運動醫學（評估後運動處方）、營養師（飲食支持）— 優先於直接藥品推廣。
